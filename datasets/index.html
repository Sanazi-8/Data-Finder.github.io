Description Regarding the 4 Databases

In the incident log data base, I created three search bars where the user can browse through the data base about the incidents information, how it happend and the date it happend

In the Urban Forest data base I created three search bars where the user can browse through the data base about the Tree name, rating and common code

In the Recycle Schedule log data base I created three search bars where the user can browse through the data base about the address , the communities name and the collection of the recycled stuff

In the police station log data base I created three search bars where the user can browse through the data base about the address of the police station, the address and the ohone number
